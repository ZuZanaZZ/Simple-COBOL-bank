# Simple simulation of a bank
## Functionality
### Simple database of clients
- System will ask you which client you'd like to appear as.

<img width="332" height="51" alt="image" src="https://github.com/user-attachments/assets/a764319b-2d3c-426a-a240-b0a76872d246" />

- After selecting a client, their information is displayed.
- You'll be asked how much you'd like to transfer to the bank.

<img width="502" height="148" alt="image" src="https://github.com/user-attachments/assets/713fadbe-e099-4a6b-999c-93611d8fed31" />

### Transaction authorisation
- Upon entering the transaction amount the system will check if there are enough funds in the account.
- If yes, the transaction will take place.

<img width="471" height="154" alt="image" src="https://github.com/user-attachments/assets/3c4aa908-fdab-48b9-aa8d-36059e3e4a30" />

- If no, the transaction is denied.
