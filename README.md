# Simple simulation of a bank
## Functionality
### Database of clients
System will ask you which client  would you like to appear as. You can also add a new client.

<img width="406" height="131" alt="image" src="https://github.com/user-attachments/assets/76814956-f1e9-4a0c-9e5d-2ba790fdad6b" />

#### Selecting a client
After selecting a client, their information is displayed.
You'll be asked how much you'd like to transfer to the bank.

<img width="557" height="134" alt="image" src="https://github.com/user-attachments/assets/b12f6d2b-6d21-45bf-a61e-85bdb8bffacb" />

#### Adding a new client
If you selected that you'd like to add a client, you will be asked to fill out the client information.
After that, you'll also be asked how much you'd like to transfer to the bank.

<img width="560" height="315" alt="image" src="https://github.com/user-attachments/assets/4ffbcb1e-4cd0-4d43-bd40-02c7c1a58b8a" />

### Transaction authorisation
Upon entering the transaction amount the program will check whether the account has sufficient funds.
If the funds are available, the transaction will take place.

<img width="565" height="158" alt="image" src="https://github.com/user-attachments/assets/ddb17596-997a-4139-9c7e-a08db538f41a" />

If not, the transaction is denied.
